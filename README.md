# Flashcard-App

## Lab 4

### App Description
The app allows the user to tap the front side of a flashcard, which holds a question, to see the card's back side, which holds the question's answer. The user can also show a new question and answer on the flashcards by entering them in a creation screen. The user can create as many flashcards as they want, browse through them and persist them even if they close the app as well. The most recent update incorporates animations and styling.
